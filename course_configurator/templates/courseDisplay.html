course display

this webpage displays a callendar for each possible combination of slot choices for the current settings.
the menu can be used to force enable (left click) or force disable (right click) a given option from the combinations.
(note that there is currently an issue where force disabled options get removed entirely.)
each calendar has it's own calendar controls, but there are buttons provided which allow limited control for all callendars simulatneously.
calendars are currently limited to only display 25 (due to performance limitations)
